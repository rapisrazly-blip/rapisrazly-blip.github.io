# ツール使い方まとめ

## 毎回使うコマンド

### LP作成
Claude Codeに話しかけるだけ：
> 「〇〇向けにLP作って」

作成されたファイルの場所：
```
C:\Users\rapis\OneDrive\デスクトップ\TEST\cc-lp-v2\lp\index.html
```

Figmaに送る場合：
1. `npm run build:figma`（cc-lp-v2フォルダで実行）
2. Figmaで「HTML to Figma」プラグイン → AI-Generated HTML → figma-ready.html を選択

---

### 案件探し（job-scout）
Claude Codeに話しかけるだけ：
> 「お仕事探してきて」

または自分でターミナルに貼る：
```bash
cd /c/Users/rapis/Downloads/job-scout/job-scout && npm run scout:sheet
```

結果はGoogleスプレッドシートの「案件一覧」に自動追加される：
https://docs.google.com/spreadsheets/d/1Os3e10qDnBtcrJQzhHqpBNXqpm0n94ZDxKts2xrlc7Y/edit

検索条件を変えたい場合 → スプレッドシートの「設定」シートを編集するだけ。

---

### PPTXスライド自動生成（pptx-agent）
Claude Codeに話しかけるだけ：
> 「スライド作って」「プレゼン作って」

または既存シートから再生成：
> 「生成して」「このシートからPPTX作って」

フォルダをClaude Codeで開いてから使う：
```bash
cd C:\Users\rapis\Downloads\pptx-agent-20260429T034346Z-3-001\pptx-agent
```

生成ファイルの場所：
```
C:\Users\rapis\Downloads\pptx-agent-20260429T034346Z-3-001\pptx-agent\output\
```

## トラブル時の対処

### Google認証エラーが出たとき
```bash
cd /c/Users/rapis/Downloads/job-scout/job-scout && npm run auth:google
```
ブラウザが開くのでログインして「続行」を押す。

### ポート3001が使用中エラーが出たとき
```bash
netstat -ano | grep ":3001"
# 表示されたPIDを使って：
taskkill //PID [PID番号] //F
```
