# 参考LP分析書

## 基本情報
- URL: https://caba-web.com/
- 一言で: シック・黒基調、番号付き理由列挙、機能アイコン網羅型

## セクション構成（順番通り）
1. ヘッダー＆ナビ — ロゴ + メニュー8項目 + SNS3種 + CTA
2. デモサイト紹介 — 個店用3パターン + グループ用2パターンのカルーセル
3. サービス説明 — 簡潔な紹介文
4. 選ばれる理由 — 8項目（番号01〜08 + タイトル + 説明文）
5. リニューアル問題提起 — 「こんな悩みありませんか？」9項目列挙
6. 機能一覧 — 20機能をアイコン+テキスト
7. 料金 — 月額22,000円の定額サブスク強調
8. フッター

## FVの組み方
- レイアウト: ヘッダー上部 + デモサイトをビジュアルアンカーとして前面に出す
- CTA: 「お問い合わせはこちら」（LINE連携）
- 信頼要素: デモサイト実績を視覚的に見せる

## カラーパレット
- メインカラー: ブラック
- アクセント: 不明（シック・ニュートラル）
- 背景トーン: 暗基調

## タイポグラフィ
- 番号付き見出し（01〜08）で視覚的階層化
- 料金などの数字を大きく強調

## 装飾要素
- カルーセル（デモサイト）
- アイコン付き機能リスト
- 番号ライン（01〜08）

## このLPから継承する具体ルール
- 「選ばれる理由」を01〜08の番号付きで見せる
- 「こんな悩みありませんか？」の問題提起セクションを入れる
- 機能・特徴をアイコン+テキストで網羅的に見せる
- 料金を大きな数字で強調する
- デモ/実績を視覚的に見せるセクションを入れる
